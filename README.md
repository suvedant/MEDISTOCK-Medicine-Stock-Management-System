# 💊 MEDISTOCK – Medicine Stock Management System

## 📌 Project Overview

MEDISTOCK is a web-based Medicine Stock Management System designed to efficiently manage medicine inventory, track stock levels, and handle user orders. It provides separate dashboards for Admin and Users, ensuring smooth management and monitoring of medical stock.

This system helps reduce manual errors, improves inventory tracking, and ensures better availability of medicines.

---

## 🚀 Features

### 👤 User Panel

* User Registration & Login
* Browse Available Medicines
* Place Orders
* View Order History
* Real-time Stock Availability

### 🛠️ Admin Panel

* Secure Admin Login
* Add / Update / Delete Medicines
* Stock Management & Alerts
* Order Management (Approve / Reject)
* User Management

## ⚙️ Installation & Setup

1. Clone the repository:

```id="p0k3sj"
https://github.com/suvedant/MEDISTOCK-Medicine-Stock-Management-System.git
```

2. Move project to XAMPP htdocs folder:

```id="aqb5h1"
C:\xampp\htdocs\
```

3. Start XAMPP:

* Apache ✔️
* MongoDB (ensure running) ✔️

4. Configure Database:

* Set MongoDB connection in config file
* Import collections (if available)

5. Run the project:

```id="l7m2ty"
http://localhost/MEDISTOCK/
```
